Las tecnologías web son el conjunto de lenguajes, herramientas, estándares y protocolos (como HTML, CSS, JavaScript, HTTP) utilizados para crear, visualizar y gestionar sitios y aplicaciones en internet. JSON - DOM - JQuery

  • JSON. JavaScript Object Notation, is a plain text format for storing and transporting data, is similar to the syntax for creating JavaScript Objects.
  • DOM. Document Object Model, is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.
  • JQuery. Is a fast, lightweight, and open-source JavaScript library designed to simplify the "client-side" scripting of HTML. Its primary philosophy is "Write Less, Do More", allowing developers to achieve complex tasks with a single line of code that would otherwise require many lines of raw JavaScript.

Desarrollo de Aplicaciones Móviles Nativas (DAMN)

Las aplicaciones nativas son programas diseñados y desarrollados específicamente para un sistema operativo particular, como iOS o Android. A diferencia de las aplicaciones web o las híbridas, las nativas aprovechan al máximo las capacidades y características únicas de cada plataforma.

Flutter - UI / Responsividad

Uno de los objetivos principales de Flutter es crear un marco que permita desarrollar aplicaciones desde una única base de código que se vean y se sientan geniales en cualquier plataforma. Esto significa que tu aplicación podría aparecer en pantallas de distintos tamaños, desde un reloj hasta un teléfono plegable con dos pantallas o un monitor de alta definición. Y tu dispositivo de entrada podría ser un teclado físico o virtual, un ratón, una pantalla táctil o cualquier otro dispositivo.

Flutter - Navegación

La navegación en Flutter se basa en la pila de rutas (stack) en la que cada pantalla se apila encima de la anterior, y el usuario puede retroceder a través de la pila de rutas para volver a pantallas anteriores. Navigator ofrece una variedad de métodos para controlar esta pila de rutas, como push, pop, pushNamed, entre otros, lo que permite a los desarrolladores de aplicaciones crear transiciones suaves y personalizadas entre pantallas.

Gestión de Estados en Flutter

Un estado son los datos de un objeto o información que puede ser leida sincrónicamente cuando el widget es creado y puede cambiar con el tiempo. Podríamos decir también, que todo lo que se guarda en el almacenamiento de la aplicación durante su funcionamiento se llama estado. Se usa cuando algunas partes de la IU cambian dinamicamente durante el tiempo de ejecución.
