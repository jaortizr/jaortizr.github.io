La v1 (081225) incluye un front-end muy sencillo, pero responsivo, creado con bootstrap, el cual se conecta a una BD para autenticar los datos de acceso proporcionados por los usuario, así también se ofrece un formulario que en caso de no tenr una cuenta se podrá conseguir por este medio. Los formualrios estan validados con justvalidate.js y la retroalimentación se presenta con sweetalert2.js. Cuando se autentica al usuario accede a una página 'reservada' que en caso de querer acceder de manera directa a ésta, se verifica la existencia de una sesión iniciada. El back-end esta soportado con apache, php y mysql que en conjunto sirven los datos/solicitudes que son hechas de manera asincrona, utilizandose dos herramientas: AJAX desde JQuery y la librería Axios y por suparte el back responde en formato JSON, el cual es tomado desde el front para procesarlo y desplegarlo según las necesidades.
