Tecnologías para el Desarrollo de Aplicaciones Web (TDAW)

Las tecnologías web son el conjunto de lenguajes, herramientas, estándares y protocolos (como HTML, CSS, JavaScript, HTTP) utilizados para crear, visualizar y gestionar sitios y aplicaciones en internet. Permiten la interacción, el intercambio de información y la funcionalidad multimedia entre usuarios y servidores a través de navegadores.

W3 Schools

Desarrollo de Aplicaciones Móviles Nativas (DAMN)

Las aplicaciones nativas son programas diseñados y desarrollados específicamente para un sistema operativo particular, como iOS o Android. A diferencia de las aplicaciones web o las híbridas, las nativas aprovechan al máximo las capacidades y características únicas de cada plataforma.
